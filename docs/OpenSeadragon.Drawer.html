Class: Drawer

OpenSeadragon. Drawer

Handles rendering of tiles for an OpenSeadragon.Viewer. A new instance is created for each TileSource opened (see OpenSeadragon.Viewer#drawer).

    addOverlay(element, location, placement, onDraw)

    Adds an html element as an overlay to the current viewport. Useful for highlighting words or areas of interest on an image or other zoomable interface.
    Parameters:
    Name Type Description
    element Element | String | Object A reference to an element or an id for the element which will overlayed. Or an Object specifying the configuration for the overlay
    location OpenSeadragon.Point | OpenSeadragon.Rect The point or rectangle which will be overlayed.
    placement OpenSeadragon.OverlayPlacement The position of the viewport which the location coordinates will be treated as relative to.
    onDraw function If supplied the callback is called when the overlay needs to be drawn. It it the responsibility of the callback to do any drawing/positioning. It is passed position, size and element.
