Class: TmsTileSource

OpenSeadragon. TmsTileSource

A tilesource implementation for Tiled Map Services (TMS). TMS tile scheme ( [ as supported by OpenLayers ] is described here ( http://openlayers.org/dev/examples/tms.html ).
