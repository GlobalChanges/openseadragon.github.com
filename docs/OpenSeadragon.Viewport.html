Class: Viewport

OpenSeadragon. Viewport

Handles coordinate-related functionality (zoom, pan, rotation, etc.) for an OpenSeadragon.Viewer. A new instance is created for each TileSource opened (see OpenSeadragon.Viewer#viewport).
